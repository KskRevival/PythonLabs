## Описание
Данное приложение является реализацией 5-ех алгоритмов поиска подстроки в строке и сравнения их по производительности и использованию памяти.

## Требования
- Python версии не ниже 3.9
- Установленная библиотека Matplotlib (для построения графиков)

## Состав
- Консольная версия: main.py
- Расположение реализации 5 алгоритмов: algos.py
- Функции для тестирования по производительности и использованию памяти: benchmark.py
- Построение графиков исходя из результатов тестирования: plotbuilder.py
- Генерация тестов: textgen.py
